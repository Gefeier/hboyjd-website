## 处理脚本(下次重新拉素材)

### 解压 PPTX media
```bash
mkdir -p "C:/Users/mac/AppData/Local/Temp/pptx_extract"
cd "C:/Users/mac/AppData/Local/Temp/pptx_extract"
unzip -o "G:/网站建设相关/欧阳聚德第6版.pptx" "ppt/media/*" -d .
```

### Slide → image 精确映射(找哪张图属于哪张 slide)
```python
from pptx import Presentation
p = Presentation(r"G:\网站建设相关\欧阳聚德第6版.pptx")
for i, slide in enumerate(p.slides, 1):
    title = next((sh.text_frame.text.strip()[:50] for sh in slide.shapes if sh.has_text_frame and sh.text_frame.text.strip()), "")
    imgs = [str(rel.target_ref or rel.target_part.partname).split('/')[-1]
            for rel in slide.part.rels.values() if 'image' in rel.reltype]
    if imgs: print(f"S{i}|{title}|{','.join(imgs)}")
```

### 批量 resize + 转 webp(模板)
```python
from PIL import Image
img = Image.open(src)
if img.mode != 'RGB': img = img.convert('RGB')
w, h = img.size
if max(w, h) > 1200:
    r = 1200 / max(w, h)
    img = img.resize((int(w*r), int(h*r)), Image.LANCZOS)
img.save(dst + '.jpg', 'JPEG', quality=85, optimize=True)
img.save(dst + '.webp', 'WEBP', quality=80, method=6)
```
